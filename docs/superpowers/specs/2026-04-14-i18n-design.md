# 国际化 (i18n) 设计

## 概述

为 Tooltip、Loot Tracker、设置面板、升级预估添加中英双语支持，使用与 `getItemName()`/`getActionName()` 相同的 `isZHInGameSetting` 模式。

## 架构

**翻译函数：**
```javascript
// src/utils.js
export function t(zh, en) {
    return globals.isZHInGameSetting ? zh : en;
}
```

**翻译文本：** 集中在 `src/i18n.js`，各文件按需引用

**使用示例：**
```javascript
// 在 HTML 中使用
t('原料', 'Material')

// 在 JS 字符串中使用
const text = t('每小时支出', 'Hourly Expenditure');
```
